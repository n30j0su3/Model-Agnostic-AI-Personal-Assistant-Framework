# Workspace Context: Development

## Visión General
Gestión de ingeniería de software, arquitectura de sistemas y experimentación técnica.

## Áreas de Trabajo
- **Backend/Frontend**: Proyectos de aplicación.
- **DevOps**: Scripts de automatización y CI/CD.
- **AI Integration**: Desarrollo de prompts y agentes.

## Estándares
- **Clean Code**: Priorizar legibilidad y mantenibilidad.
- **Documentación**: Todo proyecto debe tener un `README.md` robusto.
- **Control de Versiones**: Uso estricto de Git (Conventional Commits).

## Herramientas Preferidas
- Lenguajes: Python, TypeScript, Rust.
- Terminal: OpenCode / Claude Code.

## Archivos Clave
- `snippets/useful-functions.md`: Biblioteca de código reutilizable.
- `documentation/architecture-patterns.md`: Guía de patrones usados.
- `projects/status.md`: Trazabilidad de proyectos de software.
