# Prompt: Cierre de Sesión Diaria

Por favor, realiza las siguientes acciones para cerrar el día:

1. **Analiza SESSION.md**:
   - Cuenta tareas completadas.
   - Identifica tareas "In Progress" y "Pending".
2. **Genera Resumen**:
   - Crea un bloque de texto que resuma los logros principales.
   - Extrae las decisiones clave tomadas hoy.
3. **Prepara Mañana**:
   - Actualiza la sección `Tomorrow's Preview` con las tareas que no se terminaron.
4. **Archiva (Opcional)**:
   - Pregunta al usuario si desea que genere el archivo histórico en `sessions/YYYY/MM/DD.md` usando el template oficial.

Tono: Profesional, motivador y orientado a datos.
