# Professional Workspace

Manage your career and business with precision. This workspace enforces a more formal tone and rigorous traceability.

## Directory Structure
- `clients/`: Interaction logs and requirements per client.
- `projects/`: Professional deliverables.
- `objectives/`: OKRs and KPIs.

## Key Skills
- **@docx**: Draft contracts or formal reports.
- **@pptx**: Prepare client presentations.
- **@github-deployer**: Keep your professional portfolio updated.

## Security Rule
Never store raw API keys or secrets in this workspace. Use environment variables or placeholders.
